Zeigt die neuesten Deals von Wezar.de in einem Widget an.

== Description ==

Zeigt die neuesten Deals von Wezar.de in einem Widget an. Es können bis zu 10 Deals angezeigt werden.
Das Plugin fragt die Deals dazu von unserem Server ab. Wir speichern keinerlei Daten. Weder zu Ihrer Seite, noch zu Ihren Benutzern.

Das Plugin übernimmt das Design ihrer Seite.
Hinweis: Es werden Links zur Wezar.de Webseite angezeigt, über die sich der Benutzer weiter zu den Deals informieren kann.

== Installation ==

1. Das Plugin in den Ordner `/wp-content/plugins/wezar-stream` hochladen oder über Plugins > Installieren hinzufügen.
2. Das Plugin unter 'Plugins' aktivieren
3. Über Design > Widgets das bereitgestellte Widget zum gewünschten Ort hinzufügen
